JAVALA THEME
http://cmsthemefactory.com/themes/javala

INSTALL
1. Upload the Javala theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Javala theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. To set up the theme like the demo, please visit http://cmsthemefactory.com/javala-theme-instructions/.

WIDGET AREAS
Header Right - This is the widgeted area that appears after the title area section of the header.
Promotions - This is the section for the three homepage banner/buttons for promotions.
Homepage Slideshow - This is the homepage slideshow widget area.
Footer - These are set for three widgets in the footer area.
After Entry - This is the after entry area for single entries.
